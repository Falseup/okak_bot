## ОКАК бот (aiogram)

Телеграм-бот для групповых чатов, который реагирует на сообщения,
содержащие «окак». Базовая вероятность ответа — 15%, но есть списки
игнорируемых пользователей и «VIP», которым отправляются особые фразы.

### Настройка
- Установите зависимости: `pip install -r requirements.txt`
- Создайте бота через [@BotFather](https://t.me/BotFather)
- Откройте `config.py` и впишите токен в переменную `BOT_TOKEN`
- Заполните файлы в каталоге `data/`
  - `ignored_users.txt` — юзернеймы или ID, бот молчит
  - `special_users.txt` — всегда получают ответ
  - `special_phrases.txt` — фразы для особых пользователей

### Запуск
```
python main.py
```

Файлы в `data/` перечитываются при каждом сообщении, поэтому менять их
можно без перезапуска.
